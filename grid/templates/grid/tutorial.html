Tutorial

Step 1: Provide parameters
  • Choose to draw grid or upload text file for grid geometry (the uploaded file need to be in correct format for the parser to work correctly. For first time user, it is advised to choose draw grid option. After you are done with creating the grid, you can save it as a text file for next time use)
  • If the draw option is choosen: input number of rows (R), columns (C), channel width (w) and length (l) of grid
  • If the grid geometry is provided from file, you do not need to specify number of rows and columns of grid
  • Input diffusion coefficient of your chemicals
  • Click submit
Step 2: Provide grid geometry and inlet parameters
  • The right canvas shows an initially empty (R+2)xC grid with the top row is reserved for input and the last row for output
  • You can click on any channel to select/unselect that channel. Note that there need to be at least one inlet, one outlet and the grid has to be connected
  • After you are done drawing the grid, input inlets' velocity and concentration on the left side. The inlet number for the inlets you chose in the previous step are colored orange. Velocity unit is mm/s and should not exceed 100mm/s to guarantee the flow is still laminar. The concentration is normalized concentration with value from 0-1. Note that for choosn inlets, their concentration need to be non-increasing from left to right.
  • Click Run
Step 3: Modify geometry and inlet parameters for desired result
  • If there is an error message, fix inlet parameters and/or grid geometry then click run again. Otherwise, the result is shown on the left pannel. Also note that unnecessary channels are removed.
  • You can continue to modify the grid and inlet parameters and run again to get new result
Step 4: Save grid geometry to text file
  • Click Download Grid File to download grid geometry as mygrid.txt in your default download folder. The file has information about grid dimension and geometry.
  • You can use this text file to upload grid geometry in the 1st step
